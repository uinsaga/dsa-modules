## Soal Tugas

### Soal 1 – Variabel dan Operasi Dasar

Buat program Python yang meminta pengguna memasukkan:

- Nama mahasiswa
- Nilai tugas
- Nilai UTS
- Nilai UAS

Kemudian hitung **nilai akhir** dengan rumus:

```
Nilai Akhir = (30% Tugas) + (30% UTS) + (40% UAS)
```

Program harus menampilkan:

- Nama mahasiswa
- Nilai akhir
- Keterangan **Lulus** jika nilai ≥ 60, dan **Tidak Lulus** jika kurang dari 60.

---

### Soal 2 – Percabangan (If Statement)

Buat program untuk menentukan **kategori umur** seseorang.

Input: umur

Kategori:

| Umur    | Kategori   |
| ------- | ---------- |
| 0 – 12  | Anak-anak  |
| 13 – 17 | Remaja     |
| 18 – 35 | Dewasa     |
| > 35    | Dewasa Tua |

Output harus menampilkan kategori umur tersebut.

---

### Soal 3 – Perulangan (Loop)

Buat program Python untuk:

Menampilkan **bilangan genap dari 1 sampai 50** menggunakan perulangan.

Contoh output:

```
2
4
6
8
...
50
```

Tambahkan juga **jumlah total dari semua bilangan genap tersebut**.

---

### Soal 4 – List

Buat program yang:

1. Meminta user memasukkan **5 angka**
2. Simpan angka tersebut dalam **list**
3. Tampilkan:

- List angka
- Nilai terbesar
- Nilai terkecil
- Nilai rata-rata

Contoh output:

```
Data angka : [10, 5, 8, 20, 15]
Nilai terbesar : 20
Nilai terkecil : 5
Rata-rata : 11.6
```

---

### Soal 5 – Fungsi (Function)

Buat fungsi Python bernama:

```
hitung_luas_persegi(sisi)
```

Fungsi tersebut harus:

- menerima parameter **sisi**
- mengembalikan **luas persegi**

Gunakan fungsi tersebut dalam program utama untuk menghitung luas persegi dari input pengguna.

---

### Soal 6 – Logika Algoritma

Buat program untuk menentukan **apakah suatu bilangan adalah bilangan prima atau bukan**.

Contoh:

```
Input : 7
Output : Bilangan Prima
```

```
Input : 10
Output : Bukan Bilangan Prima
```
